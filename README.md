# deploy-coolify

Dockerfile-based deployments for [Coolify](https://coolify.io).

## Services

| Service | Path | Description |
|---|---|---|
| Moodle 5.2 | [`moodle/`](moodle/) | All-in-one Moodle + PostgreSQL + Apache |
| PostgreSQL 18.3 | [`postgresql/`](postgresql/) | Standalone PostgreSQL via Docker Compose |
